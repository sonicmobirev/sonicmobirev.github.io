Getting Started with Sonic Mobirev

Sonic Mobirev is a fan-made project with the objective of preserving old and abandoned mobile applications based on the Sonic the Hedgehog franchise.

You can download Android applications and archived content. You can also check the different mobile services and their games released throughout history.

What we do

Our goals are the following:

  • Preserve old and abandoned Sonic mobile applications (mostly Android and J2ME).
  • Give some Android applications quality of life improvements.
  • Make Android applications (and potentially J2ME applications) available to download.
  • Bring awareness to Sonic mobile applications (especially old Android ones) for better documentation.

    • What does "Mobirev" mean?

    Mobirev is a blending of multiple words. "Mobi" means "Mobile", whereas "rev" means "Revival", as the project was originally named "Sonic Mobile Revival". But "rev" also means "Revolt" or "Revolution", in reference to the Android developer verification and Google's attempt at preventing anyone from installing Android apps outside of their Play Store.

    What's coming

    This website is at its early years. The Sonic Mobirev team is currently working on more projects for the website and the Android applications.

    If you want to give help and support for the Sonic Mobirev project and its mobile applications, check out what you can do in the Help the project page.

    Download apps

    This website contains the Android applications via the ANDROID APPS, in which you can download different applications. You can select an application page from the store or via the FEATURED APPS menu on the right sidebar.

    In an application page, you can download the application, visit its website if one is present, and check numerous informations about the application via the tabs. You can also download different versions of an application (either modded or original versions).

    If you want to download an application, press the DOWNLOAD APP or ⤓ DOWNLOAD APP from the VERSIONS tab.

    App installation

    Once you have downloaded the application on your device, find the .apk file from the download folder and select it to proceed the installation. Sometimes, when trying to install an application, you may encounter an "Unsafe app blocked" message. When this occurs, press "Install anyway" to proceed.

    If the problem still persists, contact Furrican via the CONTACT tab.

    Virus/Malware reports

    If an application contains a malware according to your antivirus, it is most likely a false alarm. When this occurs, you can ignore the reports and continue the installation.

    With that said, the Sonic Mobirev team is trying its best to mitigate as much malware as possible, even those who are harmless.

    Giving Help to the project

    You can give help and support the Sonic Mobirev team and project. You can even being part of the team if you participate for the project.

    To do so, you need to do one of the following tasks.

    Application modding

    Working on mods of mobile applications to give them quality of life improvements.

    You need different software tools. It's best to mod on PC for better visibility, but you can also mod on your Android device.

    To decompile and/or modify an Android application, you need an APK editor software tool (like APK Easy Tool, APK Editor or MT Manager). For the sprites, you can change them by using a drawing software tool (like Paint.net, IbisPaint or Paint Tool SAI). And for more advanced modifications, APK editor tools can decompile the source code to Smali. If you have the programming skills and knowledge, you can change the source code to implement additional functions and modifications.

    Some applications like Sonic Advance Revival and Sonic Jump Revival has their source code decompiled for compatbility with Sketchware Pro. This allows you to mod the application by using Java code, which is easier to program than Smali. To find their source codes, check out their dedicated websites and go to the Projects menu.

    Application translation

    Working on mods of mobile applications to give them any additional languages that are not originally available. You can also retranslate languages from modded applications for better grammar accuracies.

    In most cases, the text to translate are presented as sprites on sprite sheets, or located in text files. Some text files (like .dat or .sla files) are more complicated to work on and can only be modified by using a hex editor.

    Application testing

    This task is simple: test the mobile applications to find bugs and glitches. If you encounter one, please contact Furrican via the Contact menu.

    Application preservation

    To help making a bigger library of applications, you can try to find builds of Sonic mobile applications and share them with the team or with the Keitai World community.

    If you can collect multiple versions of an application, make sure to sort the applications as much as possible to make a better collection. If you have found a forgotten application, you can upload what you have found on the Internet Archive and/or contact the Sonic Mobirev team to inform about your discovery.

    Promotion

    Bringing awareness to the Sonic Mobirev project and its mobile applications by promoting them. You can share this website or make and upload a video online of this project and their applications.

    Web development

    If you have great experience with web development, you can work on and update this website. However, you need approval from Furrican to work on this task. Contact him via the Contact menu.

    Application publication

    The Sonic Mobirev team lets you upload/publish modded applications. If you want to upload your own mods on the ANDROID APPS, please contact Furrican via the Contact menu.

    You can even take the modded applications (distributed by Furrican) from the ANDROID APPS and publish them anywhere online (except on Game Jolt and itch.io). However, make sure to credit Furrican and GdGohan as the mod developers.
